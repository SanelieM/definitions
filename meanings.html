Words and Meanings


Function:

Functions are a set of organised instructions that correspond to a certain task or specific functionality a user wants to implement in their program to get a single desired outcome. Functions allow you to reuse a code


Function call:

Function call is the process of invoking a function to execute its code. When you call a function, you are essentially asking it to perform a specific task.


Variable:

A variable is a container that stores data. It allows programmers to reuse, update or to keep track of your values. Variables are essential for storing and manipulating data in computer programs.


Data types:

Data types classify the kind of value a variable holds and define the operations that can be performed on it. The 7 data types are:

  • STRING represents a series of characters
  • REAL stores decimal numbers
  • BOOLEAN represents true or false, yes or no
  • ARRAY holds a collection of values
  • FUNCTION performs a certain task
  • OBJECT can contain built-in objects and user defined objects
  • UNDEFINED represents an uninitialized variable
  • NULL indicates the absence of any value
